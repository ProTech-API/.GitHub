Sobre o Projeto

// Contexto

Projeto desenvolvido no âmbito do API (Aprendizado por Projeto Integrado) da FATEC São José dos Campos. Integra teoria e prática com metodologias ágeis como SCRUM.

// Objetivo

Desenvolver um dashboard interativo no Power BI com dados da RAIS para apoiar a Secretaria de Desenvolvimento na análise do ecossistema produtivo regional.

// Objetivos Específicos

Mapeamento setorial Distribuição geográfica Indicadores econômicos Visualizações gráficas Interface intuitiva

// Instituição

FATEC São José dos Campos · Secretaria de Desenvolvimento como parceiro cliente · Base de dados: RAIS (Relação Anual de Informações Sociais)
