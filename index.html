Sobre o Projeto

Análise do ecossistema produtivo regional

Desenvolvido no contexto do Aprendizado por Projeto Integrado (API) da FATEC, com metodologia ágil SCRUM, entregando visualizações analíticas para apoiar a Secretaria de Desenvolvimento de São José dos Campos.

Objetivo Principal

Mapear e analisar os principais setores industriais e de serviços de São José dos Campos, com indicadores econômicos e distribuição geográfica das empresas.

Dados Utilizados

Base RAIS (Relação Anual de Informações Sociais), tratada e transformada para geração de indicadores produtivos e econômicos da região.

Produto Entregue

Dashboard interativo no Power BI com painéis analíticos para comparação de setores, visualização geográfica e indicadores de desenvolvimento regional.

Metodologia Ágil

Desenvolvimento em sprints com SCRUM — Product Owner, Scrum Master e equipe de desenvolvimento, com entrega de MVPs a cada ciclo.

Documentação

Como acessar e executar

Passos para clonar o repositório, abrir o dashboard e acessar os materiais completos do projeto.

01

Clonar o repositório

No terminal, execute git clone https://github.com/ProTech-API/.GitHub.git para baixar todos os arquivos localmente.

02

Abrir o Dashboard no Power BI

Instale o Power BI Desktop (gratuito). Abra o arquivo Dashboard ProTech - 1° Sprint.pbix na raiz do repositório para visualizar os painéis interativos.

03

Explorar os dados com Python

Os scripts de tratamento de dados rodam no Google Colab. Acesse pelo link na documentação e execute as células para reproduzir a análise da base RAIS.

04

Consultar a documentação

O arquivo Documentação.pdf contém a especificação técnica completa, o product backlog detalhado e os resultados obtidos em cada sprint.

05

Acompanhar as sprints

Os MVPs de cada sprint estão linkados na seção acima. A pasta /MVP no repositório contém os arquivos de acompanhamento de progresso.
